# Backend (Placeholder)

## Run Backend

```bash
cd backend
python -m venv .venv
```

Activate virtual environment:

- Windows (PowerShell):

```powershell
.venv\Scripts\Activate.ps1
```

- macOS/Linux:

```bash
source .venv/bin/activate
```

Install and run:

```bash
pip install -r requirements.txt
python app.py
```

## Test Endpoints

- Open `http://127.0.0.1:5000/`
- Open `http://127.0.0.1:5000/health`
